Manual Operacional: Opt TAG Pro

O Opt TAG Pro Market foi arquitetado para otimizar a gestão visual do seu varejo, combinando agilidade no ponto de venda com precisão no controle de estoque. Este manual detalha a operação estratégica de cada funcionalidade.

1. Entrada de Dados e Cadastro

• Código de Barras: O identificador universal do seu produto. Você pode digitá-lo manualmente, utilizar um leitor infravermelho conectado, ou clicar no ícone de câmera (📷) para escanear via smartphone. O botão "✕" limpa o campo instantaneamente. Se o código já existir na sua Galeria, o sistema preencherá o resto dos dados automaticamente (Auto-Fill).

• Descrição do Produto: A nomenclatura que será impressa acima do código de barras. Mantenha descrições claras e diretas. Botão "✕" disponível para correção rápida.

• Preço de Venda: Digite apenas os números (ex: 265 para R$ 2,65). O sistema formata automaticamente com a vírgula. Ao gerar a etiqueta, o símbolo monetário "R$" é inserido, garantindo padrão profissional de precificação. Botão "✕" para limpeza ágil.

2. Configuração Visual da Etiqueta

• Tipo de Etiqueta (Estratégia de Marketing):
- Normal (Branca): Padrão de precificação diária. O valor é exibido em verde corporativo, transmitindo clareza e credibilidade.
- OFERTA (Amarela): Focada em conversão. Aciona um layout de alto contraste (fundo amarelo, tarja preta "OFERTA" e preço em vermelho). Comprovado cientificamente para atrair a atenção do consumidor no PDV.

• Tamanho da Etiqueta:
- Pequena (P): Ideal para itens delicados, joias, cosméticos ou frentes de gôndola estreitas.
- Média (M): O padrão comercial para a maioria das prateleiras.
- Grande (G): Desenvolvida para fardos, caixas de atacado e identificação visual à distância.

• Quantidade: Defina quantas cópias deseja gerar de uma só vez para etiquetagem em lote.

3. Gerenciamento e Impressão

• Gerar Etiquetas: Compila os dados, salva o produto no banco de dados local (Galeria) de forma invisível e projeta as etiquetas na tela prontas para visualização.

• Imprimir: Otimiza a interface, removendo botões e menus, garantindo que apenas as etiquetas sejam processadas pela impressora. Economiza papel alinhando os itens lado a lado.

• Galeria Pessoal: Seu banco de dados integrado. Todos os produtos gerados são armazenados aqui. Clique em qualquer item para recarregá-lo instantaneamente nos campos superiores. O ícone de lixeira ("✕") remove o item do banco permanentemente.

4. Segurança de Dados (Backup)

O sistema armazena informações no cache do seu navegador. Para evitar perda de dados por formatação ou troca de máquina:
• Exportar: Gera um arquivo criptografado (JSON) com todos os seus produtos. Salve-o em um pendrive ou nuvem.
• Importar: Permite carregar o arquivo JSON previamente salvo, restaurando toda a sua galeria em qualquer outro computador ou navegador.

Em caso de falhas sistêmicas, contate o desenvolvedor via botão de suporte no rodapé.
