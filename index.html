Manual de Instruções Completo

1. Campos de Entrada (Input)

Estes são os campos onde você insere as informações do produto antes de gerar as etiquetas.

Código de Barras: O identificador único. Se você digitar ou bipar um código que já foi salvo antes, o sistema preencherá o nome automaticamente e mostrará um aviso azul no topo.
Descrição do Produto: O nome que aparecerá na etiqueta. Tente usar nomes curtos e objetivos para que a leitura seja clara na impressão.
Quantidade: Define quantas cópias daquela etiqueta específica serão criadas de uma só vez.

2. Botões de Ação Principal

Localizados logo abaixo dos campos, eles controlam o fluxo de trabalho.

➕ Gerar Tags (Botão Roxo): Processa as informações dos campos acima e as transforma em etiquetas visuais na área de impressão. Ao clicar, o sistema salva o produto na sua "Galeria" permanentemente e limpa os campos para o próximo item.
🖨️ Imprimir (Botão Verde): Abre a janela de impressão do seu dispositivo. Ele oculta automaticamente todos os botões e menus, deixando apenas as etiquetas brancas prontas para a impressora.
🗑️ Limpar Tela (Botão Vermelho): Remove todas as etiquetas que foram geradas e estão pendentes na área de impressão. Use isso quando terminar um lote e quiser começar um novo do zero.

3. Funções Adicionais

📷 Scanner (Ícone de Câmera): Ativa a câmera do celular para ler códigos físicos. Ao ler com sucesso, ele fecha a câmera sozinho e tenta buscar o nome no seu banco de dados.
🌓 Tema: Alterna entre o modo claro e o modo escuro. O modo escuro economiza bateria e é melhor para ambientes com pouca luz.
📂 Galeria de Produtos: Todos os produtos que você já gerou uma etiqueta ficam salvos aqui embaixo. Se clicar em um item da galeria, os dados são jogados para cima para você gerar etiquetas dele novamente com um toque.

⚠️ Notas Importantes

• Para apagar um produto da galeria, clique no "X" vermelho sobre ele.

• Para remover uma etiqueta individual da fila de impressão, use o "X" no canto de cada etiqueta gerada.

• O sistema salva tudo no navegador. Se você limpar os dados do navegador, a galeria será apagada.
