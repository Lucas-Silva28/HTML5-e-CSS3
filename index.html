Manual Operacional: Opt TAG Pro

O Opt TAG Pro Market foi arquitetado para otimizar a gestão visual do seu varejo, combinando agilidade no ponto de venda com precisão no controle de estoque. Este manual detalha a operação estratégica de cada funcionalidade.

1. Entrada de Dados e Cadastro

• Código de Barras: O identificador universal do seu produto. Ao digitar um código já salvo, o sistema restaura automaticamente nome, preço, tamanho e tipo de etiqueta.

• Descrição do Produto: A nomenclatura que será impressa. Mantenha descrições claras.

• Preço de Venda: Digite apenas os números. O sistema formata automaticamente.

2. Configuração Visual da Etiqueta

• Tipo e Tamanho: Ao gerar a etiqueta, estas configurações são salvas individualmente para cada produto no seu banco de dados local.

3. Gerenciamento e Impressão

• Galeria Pessoal: Clique em qualquer item para recarregá-lo instantaneamente com todas as suas configurações originais de impressão.

4. Segurança de Dados (Backup)

• Exportar/Importar: Garanta que seus produtos e configurações personalizadas estejam seguros gerando backups periódicos.

Em caso de falhas sistêmicas, contate o desenvolvedor via botão de suporte no rodapé.
