🚀 Guia de Uso

1
Cadastrar: Digite o código e nome. Ao adicionar, ele fica salvo na sua galeria para sempre.
2
Usar Atalho: Toque nos produtos da "Galeria" para gerar etiquetas sem digitar nada.
3
Scanner: Use o ícone 📷 para ler códigos com a câmera. O sistema faz um "beep" ao ler.
4
Gerar PDF: Use o botão de impressão para organizar as etiquetas em folha A4.
